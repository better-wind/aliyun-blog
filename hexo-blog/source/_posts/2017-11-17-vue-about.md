### .vue中 scope的实现

vue文件可以实现当前文件的css作用域
实现手法是通过vue-loader后生成一个
随机字符串属性 通过内置css属性选择器实现
like that
```html
<template>
    <div class="scope-div"></div>
</template>
<style scope>
    .scope-div{
        width: 200px;
    }
</style>
<!--打包处理后-->
<style>
    .scope-div[data-v-7af99ee4]{
        width: 200px;
    }
</style>
<div class="scope-div" data-v-7af99ee4></div>
```
